# SKRoot - SuperKernelRoot - Linux 完美内核级隐藏ROOT演示
新一代SKRoot，挑战全网root检测手段，跟面具完全不同思路，摆脱面具被检测的弱点，完美隐藏root功能，全程不需要暂停SELinux，实现真正的SELinux  0%触碰，通用性强，通杀所有内核，不需要内核源码，直接patch内核，兼容安卓APP直接JNI调用，稳定、流畅、不闪退。
### 本工具功能列表：
### 	1.显示自身权限信息
### 	2.获取ROOT权限
### 	3.执行ROOT命令
### 	4.执行原生内核命令
### 	5.安装部署su
### 	6.注入su到指定进程
### 	7.完全卸载清理su
